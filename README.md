# Simple Warehouse System 📦

A robust Console Application built with **C#** to manage product inventory. This project was developed to practice and demonstrate fundamental **Object-Oriented Programming (OOP)** concepts.

## 🚀 Technical Highlights
* **Encapsulation:** Protected internal data using `private set` and backing fields, ensuring that quantity and price can only be modified through specific business logic methods.
* **Constructors & Overloading:** Implemented custom constructors to ensure data integrity from the moment an object is instantiated.
* **Input Validation:** Built-in logic to prevent negative stock values and invalid product names.
* **Console UI:** An interactive loop (`while`) that allows continuous user interaction without restarting the application.

## 🛠 Technologies
* **Language:** C#
* **Platform:** .NET 8 / .NET 9
* **IDE:** Visual Studio 2022

## 📂 Project Structure
* `Entities/`: Contains the `Product` class with all the business logic and encapsulation rules.
* `Program.cs`: The entry point of the system, handling user input and the interface loop.
